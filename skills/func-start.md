# Func-Start Skill

Start the web development server.

## Usage

```
/func-start
```

## Commands

```bash
cd spotdrop-web
npm run dev
```

## What it does

1. Starts Vite dev server with HMR
2. Server runs on http://localhost:5173
3. Hot module replacement enabled

## Prerequisites

- Dependencies installed (`npm install`)
- .env file with Mapbox token
